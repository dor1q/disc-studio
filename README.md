# L805 Disc Studio

Desktop-программа для подготовки макетов под Epson L805: печать на CD/DVD, передняя обложка, задник jewel case и буклет.

Проект сделан как современная альтернатива Epson Print CD, но с отдельными макетами обложек и темной темой.

## Возможности

- Редактор диска Epson L805 с профилями `Full face 18-120 мм`, `Обычный 43-116 мм`, `Mini CD 18-80 мм`.
- Обложки jewel case: передняя `120x120 мм`, задник `151x118 мм`, буклет-разворот `240x120 мм`.
- Светлая и темная тема.
- Отдельный фон, слои и шаблоны для каждого макета.
- Текст, круговой текст, треклист, логотип/картинка как отдельные слои.
- Графические элементы: прямоугольник, эллипс, линия.
- Импорт треклиста из `CUE`, `TXT` или `CSV`.
- Эффекты фона и изображения: яркость, контраст, насыщенность, ч/б.
- Цветной фон без изображения.
- Форматирование текста: шрифт, цвет, жирность, выравнивание, курсив, подчеркивание.
- Калибровка L805: смещение X/Y, масштаб, поворот и насыщенность печати.
- Калибровочные шаблоны: проверка позиции и проверка плотности печати.
- Проверка макета перед печатью: выход за край, safe zone, скрытые и пустые слои.
- Экспорт `SVG`, `PNG 300 DPI`, печать или PDF через системный диалог.
- Desktop-режим на Electron с системными окнами открытия/сохранения.

## Быстрый запуск

Готовая Windows-сборка находится в папке:

```text
L805 Disc Studio-win32-x64/L805 Disc Studio.exe
```

Для запуска из исходников:

```bash
npm install
npm start
```

## Сборка Windows-версии

```bash
npm install
npm run package:win
```

Сборка появится рядом с папкой проекта:

```text
../L805 Disc Studio-win32-x64/
```

## Рабочий порядок

1. Вверху холста выбери `Макет`: диск, передняя обложка, задник или буклет.
2. Нажми `Фон` и загрузи изображение для текущего макета.
3. Нажми `Альбом` для шаблона диска или `Обложка` для шаблона выбранной обложки.
4. Добавь текст, треклист или логотип, затем настрой координаты, размер, цвет и прозрачность.
5. Для CUE/TXT/CSV нажми `CUE/TXT` и выбери файл.
6. Добавь `Фигура`, если нужен графический элемент без картинки.
7. При необходимости настрой эффекты фона или изображения.
8. Нажми `Проверить`, чтобы увидеть предупреждения перед печатью.
9. Для диска используй `Тест`, чтобы распечатать калибровочную сетку L805 и внести X/Y поправку.
10. Используй `Плотность`, чтобы подобрать насыщенность печати для конкретной марки диска.
11. Для вывода нажми `Печать`, `PDF`, `SVG` или `PNG`.

## Печать на Epson L805

Для диска в диалоге печати выбирай Epson L805, источник `CD/DVD Tray`, масштаб `100%`, без подгонки под страницу.

Если поверхность диска мажется или остается липкой после высыхания, снизь `Насыщенность печати` в блоке калибровки.

Кнопка `Плотность` печатает проверочный шаблон с несколькими уровнями насыщенности. Выбери самый чистый вариант без смазывания и выставь близкое значение в `Насыщенность печати`.

Для обложек печатай на обычной бумаге/фотобумаге с масштабом `100%`. Размер страницы в окне печати выставляется по выбранному макету.

## Горячие клавиши

- `Ctrl+Z` / `Ctrl+Y` - отмена и повтор.
- `Delete` / `Backspace` - удалить выбранный слой.
- Стрелки - сдвиг выбранного объекта на `0.2 мм`.
- `Shift` + стрелка - сдвиг на `1 мм`.
